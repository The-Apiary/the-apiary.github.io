VocalEyes

Interface

  1. Use the spacebar to select a group of characters.
  2. Within a group of characters, select one to add it to the Word Builder.
  3. Select _ to add the word in the Word Builder to the Sentence Builder.
  4. The Sentence Builder inserts spaces between words.
